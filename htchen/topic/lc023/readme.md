# 23.合并K个升序链表

## 1. 题目描述

给你一个链表数组，每个链表都已经按升序排列。

请你将所有链表合并到一个升序链表中，返回合并后的链表。

 

 **示例 1：** 

```
输入：lists = [[1,4,5],[1,3,4],[2,6]]
输出：[1,1,2,3,4,4,5,6]
解释：链表数组如下：
[
  1->4->5,
  1->3->4,
  2->6
]
将它们合并到一个有序链表中得到。
1->1->2->3->4->4->5->6

```
 **示例 2：** 

```
输入：lists = []
输出：[]

```
 **示例 3：** 

```
输入：lists = [[]]
输出：[]

```
 

 **提示：** 
-  `k == lists.length` 
-  `0 <= k <= 10^4` 
-  `0 <= lists[i].length <= 500` 
-  `-10^4 <= lists[i][j] <= 10^4` 
-  `lists[i]` 按 **升序** 排列
-  `lists[i].length` 的总和不超过 `10^4` 
 
**标签**
`链表` `分治` `堆（优先队列）` `归并排序` 


## 2. 解题
1.采用顺序合并的方法，先创建一个空的结果链表，依次将该链表与下一个链表合并，将结果存入结果链表中。
2.采用归并排序的思想，每轮将两个链表进行合并，直至合并成一个结果链表。
