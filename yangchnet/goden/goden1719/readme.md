# 面试题 17.19.消失的两个数字

## 1. 题目描述

给定一个数组，包含从 1 到 N 所有的整数，但其中缺了两个数字。你能在 O(N) 时间内只用 O(1) 的空间找到它们吗？

以任意顺序返回这两个数字均可。

**示例 1:**

```
输入: [1]
输出: [2,3]
```

**示例 2:**

```
输入: [2,3]
输出: [1,4]
```
**提示：**

- `nums.length <= 30000`

**标签**

`位运算` `数组` `哈希表`


## 2. 解题

数组后补两个-1，使用抽屉原理对数组进行排序，O(n)的时间复杂度，再遍历一遍数组找到两个-1的位置就是答案
