# 面试题 10.09.排序矩阵查找

## 1. 题目描述

给定MxN矩阵，每一行、每一列都按升序排列，请编写代码找出某元素。

 **示例:** 

现有矩阵 matrix 如下：

```
[
  [1,   4,  7, 11, 15],
  [2,   5,  8, 12, 19],
  [3,   6,  9, 16, 22],
  [10, 13, 14, 17, 24],
  [18, 21, 23, 26, 30]
]

```
给定 target = `5` ，返回 `true` 。

给定 target = `20` ，返回 `false` 。


**标签**
`数组` `二分查找` `分治` `矩阵` 

## 2. 解题
将矩阵看作一个二叉树，根节点为右上角节点。
