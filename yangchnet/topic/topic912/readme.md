# 912.排序数组

## 1. 题目描述

给你一个整数数组 `nums` ，请你将该数组升序排列。

 
 **示例 1：** 

```

输入：nums = [5,2,3,1]
输出：[1,2,3,5]

```
 **示例 2：** 

```

输入：nums = [5,1,1,2,0,0]
输出：[0,0,1,1,2,5]

```
 

 **提示：** 
-  `1 <= nums.length <= 5 * 10^4` 
-  `-5 * 10^4 <= nums[i] <= 5 * 10^4` 
 
**标签**
`数组` `分治` `桶排序` `计数排序` `基数排序` `排序` `堆（优先队列）` `归并排序`
