# 括号生成

## 1. 题目描述
数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。

有效括号组合需满足：左括号必须以正确的顺序闭合。

## 2. 示例
示例1
```
输入：n = 3
输出：["((()))","(()())","(())()","()(())","()()()"]
```

示例2
```
输入：n = 1
输出：["()"]
```

**提示**
- 1 <= n <= 8

## 3. 解题
分别使用left,right来表示还未加入字符串的左右括号，要求left>0, right>=left
使用递归回溯法往字符串中添加括号，当l=r==0时，将字符串添加到数组中。递归完成后，即得到结果。
