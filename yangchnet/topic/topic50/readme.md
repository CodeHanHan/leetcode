# 50.Pow(x, n)

## 1. 题目描述

实现 <a href="https://www.cplusplus.com/reference/valarray/pow/" target="_blank">pow( *x* , *n* )</a> ，即计算 `x` 的整数 `n` 次幂函数（即， `x^n` ^<span style="font-size:10.8333px"> </span>）。



 **示例 1：**

```

输入：x = 2.00000, n = 10
输出：1024.00000

```
 **示例 2：**

```

输入：x = 2.10000, n = 3
输出：9.26100

```
 **示例 3：**

```

输入：x = 2.00000, n = -2
输出：0.25000
解释：2^-2 = 1/2^2 = 1/4 = 0.25

```


 **提示：**
-  `-100.0 < x < 100.0`
-  `-2^31 <= n <= 2^31-1`
-  `-10^4 <= x^n <= 10^4`

**标签**
`递归` `数学`
