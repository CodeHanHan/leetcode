# 加一

## 1.题目描述
给定一个由整数组成的非空数组所表示的非负整数，在该数的基础上加一。  
最高位数字存放在数组的首位，数组中每个元素只存储单个数字。  
你可以假设除了整数 0 之外，这个整数不会以零开头。  


## 2. 示例
提示：
```
1 <= digits.length <= 100
0 <= digits[i] <= 9
```
示例 1：
```
输入：digits = [1,2,3]
输出：[1,2,4]
解释：输入数组表示数字 123。
```
示例 2：
```
输入：digits = [4,3,2,1]
输出：[4,3,2,2]
解释：输入数组表示数字 4321。
```
示例 3：
```
输入：digits = [0]
输出：[1]
```

## 3. 解题
从后向前遍历，若最后一位小于9，直接加1，结束；  
若等于9，则存在进位问题，将最后一位变成0，按照此方法继续向前遍历；  
若各位全为9，则需要增加一位，将第一位为1，其余全为0。
